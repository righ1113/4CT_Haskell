# 四色定理とは
四色定理（よんしょくていり／ししょくていり、英: Four color theorem）とは、  
厳密ではないが日常的な直感で説明すると  
「平面上のいかなる地図も、隣接する領域が異なる色になるように塗り分けるには4色あれば十分だ」という定理である。 

グラフ理論でとらえると、  
- 「平面グラフは4彩色可能である」  
という定理になる（後述）  

（Wikipediaより）
